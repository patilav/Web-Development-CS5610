Experiment 11 : JQuery Fade In/Out Effects

About the experiment

jQuery Fading Methods

With jQuery you can fade an element in and out of visibility. jQuery has the following fade methods:
  • fadeIn()
  • fadeOut()
  • fadeToggle()
jQuery fadeIn() Method
The jQuery fadeIn() method is used to fade in a hidden element.
jQuery fadeOut() Method
The jQuery fadeOut() method is used to fade out a visible element.
jQuery fadeToggle() Method The jQuery fadeToggle() method toggles between the fadeIn() and fadeOut() methods.
If the elements are faded out, fadeToggle() will fade them in.
If the elements are faded in, fadeToggle() will fade them out.

Code Snippet

$(document).ready(function () {
$("#HideButtton").click(function () {
$("#div1").fadeOut();
$("#div2").fadeOut("slow");
$("#div3").fadeOut(3000);
});
$("#ShowButtton").click(function () {
$("#div1").fadeIn();
$("#div2").fadeIn("slow");
$("#div3").fadeIn(3000);
});
});

Here I have used fadeIn and fadeOut effects to manipulate the effects where
if given no argument the fadeIn/fadeOut even will occure immediatedly.
if given slow as argument the fadeIn/fadeOut even will occure slowly.
if given the interval 3000 as argument the fadeIn/fadeOut even will occure slowly after 3000 milliseconds.
