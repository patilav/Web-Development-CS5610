Experiment 10: 2D Animation using JQuery

About this experimet

This experiment showcases simple JQuery animation which can be used anywhere to make the page attractive. I have animated the changes in size, opacity and border of an element with class name "experiment-block" after clicking on "GO" button

Jquery code snippet:

<script>{
$("#go").click(function () {
$("#block").animate({
{{
opacity: 0.6,{
marginLeft: "0.8in",{
fontSize: "2em",{
width: "50%",{
}, 1500);{
});{
</script>{
