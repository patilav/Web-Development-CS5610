app.delete("/api/website/:id", function (req, resp) {
websites.splice(req.params.id,1);
resp.json(websites);
});

Description :
Above code snippet will change / delete the a json document on the server.
where the websites will remove the row with given index and data is sent back using http get method to display on the page. The above code snippet shows that we can use app.delete to delete the content from website. the website.splice will remove the only 1 content starting with given index in the code. For example : "Site3" is removed from key and pair
