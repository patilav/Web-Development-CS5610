Experiment 26 : Simple Node.js Response

About the experiment

Node.js is the cross-platform runtime envirnment for server-side and networking application . In this experiment I have just returned simple text after following steps :
1) Register application on Open Shift .
2) Create an new node.js application.
3) Installed nodeJS and it's dependancies, git, rhc.

Nodejs code snippet view

Simple Node.js : Code Snippet View:

app.get('/content', function (req, res) { res.send('<h1>Hello from Open Shift - Static Message</h1>'); });

Description :
Above code snippet will send a simple response to the client who have accessed the link.
1) app.get is supported http method in node.js ,app.get will send responce to the client .
2) When a server hits http://cs5610-avantipatil.rhcloud.com/ link it will search for /content in js according to the content in the method , js will send responce.

Required Libraries

Express is the framwork used here which is declared as required, require('express') will import the require library to run the application using Express framework .
