Experiment 9: JQuery Basic

About the experiment

This experiment demonstrate the concept of hide and show objects on button click event.
Click on the hide button to hide the image and see the div shrink!
Again, to see the image again, click the show button and watch it expand!

JQuery: code snippet view

$(document).ready(function () {
$("#hide").click(function () {
$("p").hide(1000);
});
$("#show").click(function () {
$("p").show(1000);
});
});
