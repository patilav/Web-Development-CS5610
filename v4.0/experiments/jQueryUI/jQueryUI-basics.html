Experiment 16: JQuery UI Basics : Sortable and Draggable

Draggable

Functionality:

It allows you to drag elements using the mouse. Enable draggable functionality on any DOM element. Move the draggable object by clicking on it with the mouse and dragging it anywhere within the viewport.

Options

grid: Array
Snaps the dragging helper to a grid, every x and y pixels. The array must be of the form [ x, y ].

connectToSortableType: Selector
Allows the draggable to be dropped onto the specified sortables. If this option is used, a draggable can be dropped onto a sortable list and then becomes part of it. Note: The helper option must be set to "clone" in order to work flawlessly. Requires the jQuery UI Sortable plugin to be included.

Events

start: Type: dragstart
start( event, ui )
Triggered when dragging starts.
event (Type: Event)
ui (Type: Object)
helper
Type: jQuery
The jQuery object representing the helper that's being dragged.
position
Type: Object
Current CSS position of the helper as { top, left } object.
offset
Type: Object
Current offset position of the helper as { top, left } object.
stop: Type: dragstop
stop( event, ui )
Triggered when dragging starts.
event (Type: Event)
ui (Type: Object)
helper
Type: jQuery
The jQuery object representing the helper that's being dragged.
position
Type: Object

Resizable

Functionality:

It allows you to Change the size of an element using the mouse. Enable any DOM element to be resizable. With the cursor grab the right or bottom border and drag to the desired width or height.

Sortable

Functionality:

Reorder elements in a list or grid using the mouse. nable a group of DOM elements to be sortable. Click on and drag an element to a new spot within the list, and the other items will adjust to fit. By default, sortable items share draggable properties.

Code Snippet

$(function () {
$("#sortable").sortable({ //this enables the sorting
revert: true
});
$("div.dragme").draggable({ //this enables the dragging event
connectToSortable: "#sortable",
helper: "clone",
revert: "valid",
grid: [20, 20],
start: function () { console.log("Drag start") },
stop: function () { console.log("Drag stop") }
});
$(".header").html("Adjustable Menu Bar");
$(".resizeme").resizable(); //this enables the resize ability
})
