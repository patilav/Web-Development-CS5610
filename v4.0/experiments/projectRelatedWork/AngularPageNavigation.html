About the experiment

This page displays various types of navigation bars and changing the values of the number of navigation pages which are currently being accessed and this can be reflected not just one but to all the pages which use the same variable. i.e. currentPage and bigCurrentPage


Code Snippet: 
            $scope.setPage = function (pageNo) {
                $scope.currentPage = pageNo;
            };
            $scope.pageChanged = function () {
                console.log('Page changed to: ' + $scope.currentPage);
            };

When user clicks on the link which it wants to go the page will be navigated to that perticular page.
