Experiment 1: Carousel: Main Slider information navogation

Description

Types of artwork for the projects have been presented in a carousel-based interface defined in the Bootstrap framework. The type of artwork is used as the background text for a slide, with the main area composed of all other information associated with the features available. Navigation is made using arrow keys to the left and right or the bubbles at the bottom of a slide. In addition to these, automatic transition occurs after a fixed interval.

Documentation

Carousels

A carousel is created using the specified 'carousel' and 'slide' classes. An interval has been provided to indicate the time elapsed before slide transition takes place on its own. Slides are added using the 'carousel-inner' class containing individual item divisions ('div' tags). Controls are defined by utilizing the 'data-slide' property and in-built glyphicons are used to represent the actual control items.

Code snippet:

<div id="myCarousel" class="carousel slide" data-interval="3000" data-ride="carousel">
<ol class="carousel-indicators">
<li data-target="#myCarousel" data-slide-to="0" class="active"></li>
<li data-target="#myCarousel" data-slide-to="1"></li>
<li data-target="#myCarousel" data-slide-to="2"></li>
</ol>
