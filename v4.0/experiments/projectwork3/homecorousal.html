Experiment 2: Home of project

Description

Home window of the project has been presented in a carousel-based interface defined in the UI Bootstrap framework. The type of artwork is used as the background text for a slide, with the main area composed of all other information associated with the features available. Navigation is made using arrow keys to the left and right or the bubbles at the bottom of a slide. In addition to these, automatic transition occurs after a fixed interval.

Demo Check the link to see the home page

Documentation

Carousels

A carousel is created using the specified 'carousel' and 'slide' classes. An interval has been provided to indicate the time elapsed before slide transition takes place on its own. Slides are added using the 'carousel-inner' class containing individual item divisions ('div' tags). Controls are defined by utilizing the 'data-slide' property and in-built glyphicons are used to represent the actual control items.

Code snippet:

            <div style="height: 305px">
                            <carousel interval="myInterval">
                                <slide ng-repeat="slide in slides" active="slide.active">
                                    <img ng-src="{{slide.image}}" style="margin:auto;">
                                    <div class="carousel-caption">
                                        <h4>Slide {{$index}}
                                        <p>{{slide.text}}

                                    </div>
                                </slide>
                            </carousel>
                        </div>
                        <div class="row">
                            <div class="col-md-6">
                                <button type="button" class="btn btn-info" ng-click="addSlide()">Add Slide
                            </div>
                            <div class="col-md-6">
                    Interval, in milliseconds: <input type="number" class="form-control" ng-model="myInterval">
                    

                    Enter a negative number or 0 to stop the interval.
                            </div>
                        </div>
